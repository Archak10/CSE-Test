3. House Price Prediction

This is a machine learning-based web application that predicts the price of a house based on input features like:

  • Location
  • Number of Bedrooms
  • Number of Bathrooms
  • Parking
  • Stories
  • Furnishing Status

The model is deployed using streamlit.
